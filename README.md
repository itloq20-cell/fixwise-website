FixWise Website

Project Description:
FixWise is a smart device repair platform that helps users easily book repair services.

Business Problem:
Many customers face delays and unclear pricing when repairing their devices. This causes frustration and lack of trust in repair shops.

Proposed Solution:
FixWise allows users to:
- Select device type (phone, laptop, TV)
- Describe their issue
- View estimated price
- Book repair service easily

Pages:
- Home Page
- Login / Signup
- Phone Repair
- Laptop Repair
- TV Repair
- Technician Team Page
- Cart Page
- Booking details Page

Tools Used:
- HTML & CSS
- FlutterFlow (used for UI/UX design only)
